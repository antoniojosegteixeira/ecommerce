## Getting Started
- Clone the repository
- Run npm install
- Create an .env file at the root of the directory and insert the passwords:
    - MONGODB_URI="mongodb+srv..."
    - JWT_SECRET=...
    - JWT_EXPIRE=86400
    - PAYPAL_CLIENT_ID=...
- Start with npm run dev
